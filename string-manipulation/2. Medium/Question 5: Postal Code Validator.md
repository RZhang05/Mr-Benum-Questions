# Postal Code Validator

Write a program that will check if a given Postal Code is valid, meaning it is of the form `LNL NLN`, where `L` is a letter, and `N` is a number.

## Sample Input 1
```
F2N 3S9
```
## Sample Output 1
```
Valid
```
## Sample Input 2
```
923 LFP
```
## Sample Output 2
```
Invalid
```
