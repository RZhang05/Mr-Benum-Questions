# Palindrome Tester

Your program should take an input text, and then test if the word is a **palindrome**. A **palindrome** word is one that is spelt the same forward or backwards.

## Sample Input 1
```
Kayak
```
## Sample Output 1
```
is palindrome
```
## Sample Input 2
```
hello
```
## Sample Output 2
```
not palindrome
```
